The Menu Bar

The menu bar occupies the top end of the main screen. It provides 2 or 3 drop-down lists that allows users to focus on a subset of data. For example, the status filter allows you to focus on cases that have been assigned to pathologists for diagnosis, the speciality filter can be used to add a second filter, for example to look at GI cases assigned to pathologists, and the codes filter can be used to add a third filter, for example to look at GI cases assigned to pathologists that are in the red for delayed reports. The filters are additive. Of note, only the table widget can respond to all 3 filters. The pie chart can respond to any 2 combinations of the 3 filters. The workflow and workload charts only respond to the status and speciality filters, and the turnaround gauge only responds to the specialty filter is ignored.
